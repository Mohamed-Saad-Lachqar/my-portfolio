# 🚀 My Portfolio

A modern and interactive developer portfolio built to showcase my projects, skills, and experience in web development.

This portfolio focuses on **creative front-end development**, combining smooth animations, immersive 3D graphics, and modern scrolling techniques to create an engaging user experience.

---

## ✨ Features

* Modern and responsive design
* Smooth scrolling powered by **Lenis**
* Advanced animations using **GSAP**
* Interactive **3D scenes** with React Three Fiber
* Custom **WebGL / GLSL shader effects**
* Project showcase section
* Clean and minimal UI
* Fully responsive layout

---

## 🛠 Tech Stack

* **React**
* **GSAP**
* **Lenis (Smooth Scrolling)**
* **React Three Fiber**
* **Three.js**
* **WebGL / GLSL Shaders**
* **Tailwind CSS**
* **JavaScript (ES6+)**

## ⚡ Installation

Clone the repository

git clone https://github.com/Mohamed-Saad-Lachqar/my-portfolio.git

Navigate to the project directory

cd my-portfolio

Install dependencies

npm install

Start the development server

npm run dev
